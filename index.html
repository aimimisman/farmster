Farmster

Frontend prototype for the farm marketplace system

Welcome to Farmster

This repository contains the frontend modules for the Farmster system. Each team member should work only in their assigned module folder.

Team Modules

  • Member A: Marketplace, Product Catalog, Market Price
  • Member B: Farmer Profile, Location, Certification / Product Label
  • Member C: Search and Filter, Available Products Detail
  • Member D: Chat / Messaging, Notification
  • Member E: Farm Adoption / Crowdfunding, Payment, Transaction Coordination

Project Notes

Use GitHub Pages for static frontend deployment. Backend integration with Google Apps Script can be connected later through web app endpoints.
